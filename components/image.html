Image

Images are an integral part of the content strategy of any website. The best images are set above the fold and act as both eye candy and informational tools.

Responsive Image

Responsive Image Components are used to display Images on websites, which are screen responsive. They expand upto the width of the container they are in. Use class img-responsive to make your image responsive.

Round Image

Round Images are the circular images. Use class img-round to make your image round along with img-responsive class to make it responsive as well.
